Linux初涉

发表于 2015-10-10   |   分类于 Linux   |  

说的通俗点，那是为了别人看你的眼光透着羡慕。我想每一个程序员都想像电影里的黑客一样，在命令行中噼里啪啦敲一大丢你看不懂的命令，然后，稀里哗啦的显示一大丢你貌似看得懂的东西。说的文艺点，那是为了梦想，为了去圆当初吹过的牛逼，就要花一些别人睡觉的日日夜夜去苦逼的”望闻问切”,有时候，目的也许只是为了得到一个肯定，证明自己也是可以做到别人能做到的事，达到别人曾达到的高度。

当初从瘟都死机子，到Mac机，然后初涉Linux，最后因为工作原因，又回到瘟都死，操作系统孰强孰弱，每个人都有各自的看法。作为Mac和Chrome的狂热爱好者，就不提鄙陋之见了。好了，叨叨了一圈，直奔主题吧。

环境：Mac Terminal + Linux CentOS + Mou

linux 常用命令

/ 表示root
. 表示当前目录
.. 表示当前目录的父目录

文件处理命令

ls (list)

ls -[ald] [文件或目录名称]

-a(all) 显示所有文件，包括隐藏文件
-l(long) 详细信息显示
-d(directory) 查看目录属性
ls -l 可简化为ll

-rw-r–r–

常见文件类型

‘d’ -目录directory
‘-’ -二进制文件
‘l’ -link软链接文件

r - read 读
w - write 写
x - execute 执行
所有者u - user,owner
所属组g - group
其他人o - others

cd (change directory)

cd [目录]

pwd (print working directory)

mkdir (make directories)

创建空目录

touch

创建空文件

cp (copy)

cp -R [源文件或目录][目的文件]

-R 复制目录，复制文件无须任何参数

mv

mv [源文件或目录] [目的目录]

rm (remove)

rmdir 删除空目录
rm -r [文件或目录]

-r 删除目录
-i 询问
-f 强制删除

mkdir

-m ，–mode，设定权限模式
-v ，显示信息

改变文件权限

chmod xxx 文件或目录名称

r - 4 w - 2 x - 1
chmod u + rw 增加 [文件或目录名称]
chmod o – x 减少 [文件或目录名称]
chmod g = rwx 赋予权限 [文件或目录名称]

改变文件或目录的所有者

chown [用户][文件或目录]

改变文件或目录的所属组

chgrp [组名][文件或目录]

umask -S

cat (contacenate and display files)

cat [文件名]

适合查看文件内容不是很长的文件

more

more [文件名]

分页显示内容
（空格）或f 显示下一页
Enter 显示下一行
Q或q 退出

head

查看文件前几行
head -num [文件名]
head -20 /etcrvices

tail

tail -num [文件名]

-f 动态显示

ln （link） 产生软链接

ln -s [源文件][目标文件]

软链接文件的权限 lrwxrwxrwx 并有一个箭头指向源文件
su – 用户名 切换用户

添加用户

useradd  xxx
passwd  xxx

硬链接 

ln [源文件][目标文件]

文件搜索命令

which

which [命令名称]
which  ls

whereis

whereis mysql

find

find [path][options][expression]

find命令的作用是在目录中搜索文件，它的使用权限是所有用户。

find / -name httpd.conf
find /etc -name ‘ttpd‘
-name 文件名字
-size 文件大小

  • 超过xxx大小的。单位是数据库，512字节 = 0.5kb
  • 小于
    100M = 204800
    -user 根据文件所有者查找
    以天/分钟为单位
    -ctime -cmin
    -atime -amin
    -mtime -mmin
    c-change改变，表示文件的属性被修改过（所属组，所有者权限）
    a-access访问，
    m-modify修改，文件内容被修改过
    -之内
    +超过
    find /etc –mmin -120
    在etc下找出两小时内内容被修改过的文件

    连接符

    -a and逻辑与
    -o or逻辑或
    -exec 跟上命令 {} \;
    -ok 询问确认
    {}表示find查询的结果 \表示转义符 ;表示结束
    ?匹配单个字符
    匹配任意字符
    find /etc –name init
    find /etc –name init???

    locate

    （只在系统文件目录的数据库，配合updatedb查找）
    locate services

    grep

    grep [指定字符串][源文件]
    在文件中搜索字符串匹配的行并输出
    grep ftp /etcrvices

    帮助命令 man

    man [命令或配置文件]
    查看ls命令用法
    man ls
    查看文件帮助，只能写配置文件的帮助
    man services
    info和man一样

    压缩解压缩

    tar

    tar [主选项+辅选项] 文件或目录

-v 详细报告tar处理的文件信息。如无此选项，tar不报告文件信息。
-c 创建新的档案文件。如果用户想备份一个目录或是一些文件，就要选择这个选项。
-f 使用档案文件或设备，这个选项通常是必选的。
-z 用gzip来压缩/解压缩文件，加上该选项后可以将档案文件进行压缩，但还原时也一定要使用该选项进行解压缩。

解包
tar xvf FileName.tar
打包
tar cvf FileName.tar DirName
打包后的后缀是.tar

解压
tar zxvf FileName.tar.gz
压缩
tar zcvf FileName.tar.gz DirName
打包后的后缀是.tar.gz

gzip

gzip 选项 [文件]

压缩文件，压缩后的文件格式是.gz，有两个特点
1.只能压缩文件，不能压缩目录
2.不保留原文件
gunzip 或者gzip -d 解压缩
file xxx 可查看xxx是什么文件类型

zip

zip 选项[-r] [压缩后文件]

压缩文件
zip service.zip /etcrvices
压缩目录
zip –r test.zip /test

解压缩 unzip 压缩文件或目录名字

bzip2

bzip2 选项[-k] [文件名称]

压缩文件，压缩后的后缀是.bz2
解压缩
bunzip2 选项[-k] [压缩文件]
-k 是否保留原文件

网络通信指令

ping

测试网络连通性
ping [选项] IP地址
-c count ping 多少次

ifconfig

-a 显示所有网卡信息

history

history 显示命令列表 上下箭头查看
(!序号) 执行序号的命令
ctrl+l 清屏 ctrl + u 删除光标前所有字符

别名

命名别名
alias 命令别名 = 原命令
alias copy = cp
alias drm = “rm –rf”
干掉别名
unalias 别名名称
unalias drm

输入/输出重定向

0 标准输入(STDIN)
1 标准输出(STDOUT)
2 标准错误输出(STDERR)

输出重定向（显示不在屏幕，可以显示在文件中）
ls –l /tmp > /tmp.msg

追加

输入重定向
wall < /etc/greet.txt

错误输出重定向
cp –R /usr /backup/usr.bak 2> /bak.error

管道符

ls –l /etc | grep yum | wc –l

结合上述
write < /home/joe/hello.txt && rm /home/joe/hello.txt

ps 进程管理

-a 显示所有用户的进程
-u 显示用户名和启动时间
-x 显示没有控制终端的进程
-e 显示所有进程
-l 长格式显示
-w 宽行显示，可以使用多个w进行加宽显示

查看所有用户执行的进程的详细信息
ps –le 或者 ps –aux
排序
ps –le – sort pid
查看进程树，父子进程
pstree | more

kill 关闭进程

kill 进程号

（强行关闭） ==stop
kill -9 进程号
结束所有进程

killall
（重启） ==restart
kill -1 进程号

以apache为列，查询，重启，关闭
pgrep httpd
kill -1 pgrep httpd
pkill httpd
